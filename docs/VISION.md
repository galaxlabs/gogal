# Gogal Vision

Gogal is a Go-based Low-Code / No-Code Platform + Cloud BaaS + Local-First App Generator.

## Direction
Schema -> Migration -> Dynamic CRUD API -> Server-rendered Desk -> Workflow/Permissions -> SDK/App Generators -> Cloud/Local Sync.

## Not First
Do not build ERP-complete modules first.
Prioritize a clean metadata-driven core MVP.
