# Gogal Roadmap

## Phase 0 Foundation
Name lock, architecture baseline, CLI foundation, Postgres privilege tooling.

## Phase 1 Core MVP
DocType/Field metadata, migration engine, dynamic CRUD, Desk list/form.

## Phase 2 Permissions + Workflow
Roles, permissions, workflow states/actions, server hooks.

## Phase 3 CLI + App System
Bench/site/app workflows, install/migrate/export/import/backup.

## Phase 4 App Generator
Generate app clients (Vanilla/Svelte/Vue/React/Angular/Next/Astro).

## Phase 5 Cloud Console
Tenant/project dashboards, tokens, backups, logs, generation controls.

## Phase 6 Offline Sync
SQLite local runtime, sync/conflict resolver.

## Phase 7 Advanced Business/ERP
Accounts, inventory, CRM, HR, POS, reporting, automation.
